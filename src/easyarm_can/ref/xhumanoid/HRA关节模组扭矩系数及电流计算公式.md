# HRA关节模组扭矩系数及电流计算公式

来源：用户提供的《HRA关节扭矩系数及电流计算公式_20251224》截图。

## 计算公式

按表中“额定扭矩扭矩系数”估算：

```text
输出扭矩(Nm) = 电流(A) * 额定扭矩扭矩系数(Nm/A)
电流(A) = 目标输出扭矩(Nm) / 额定扭矩扭矩系数(Nm/A)
```

若使用峰值扭矩系数，则将公式中的额定扭矩扭矩系数替换为峰值扭矩扭矩系数。

## 代码对应关系

`easyarm_can` 中 `MotorModel::torque_constant_nm_per_a` 对应本表的“额定扭矩扭矩系数 (Nm/A)”。

XHumanoid 反馈报文给出的是当前电流，驱动中按下式换算为反馈扭矩：

```text
feedback.torque_nm = current_a * torque_constant_nm_per_a
```
