# XHumanoid CAN 协议记录

此文件用于记录 XHumanoid 电机 CAN 协议细节。

## 待确认内容

- 标准帧、扩展帧或 CAN FD。
- 仲裁段和数据段波特率。
- CAN ID 编码方式，以及如何从反馈帧提取 `motor_id`。
- 使能电机帧格式。
- 失能电机帧格式。
- 清故障帧格式。
- 位置、速度、力矩、电流控制帧格式。
- 参数读写机制，以及是否存在类似 CANopen SDO 或私有参数索引。
- 反馈帧数据区格式。
- 位置单位与原始值换算关系，目标单位为 `rad`。
- 速度单位与原始值换算关系，目标单位为 `rad/s`。
- 力矩单位与原始值换算关系，目标单位为 `Nm`。
- 温度单位与原始值换算关系，目标单位为 `degC`。
- 故障码位定义。

## 实现约定

- 接收线程只负责收 CAN 帧和更新反馈缓存，不直接执行控制逻辑。
- 反馈缓存按 `motor_id` 索引。
- 协议字段确认前，不猜测硬件命令帧，避免产生不可预期的电机行为。
